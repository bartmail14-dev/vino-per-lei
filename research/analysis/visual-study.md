## DESIGN BESLISSINGEN VOOR VINO PER LEI

### Kleurenpalet (Final)
```css
/* Primary */
--wine-burgundy: #722F37;
--wine-dark: #5A252C;

/* Accents */
--gold: #C9A227;

--cream: #FAF9F7;

--charcoal: #1A1A1A;
--dark-bg: #25282A;

/* Functional */
--success: #22C55E;
--error: #EF4444;
```

### Typography
- **Logo/Brand:** Custom serif of Playfair Display
- **Headlines:** Playfair Display, various weights
- **Body:** Inter of DM Sans
- **Prijzen:** Tabular figures, semi-bold

### Must-Have Components

1. **Age Gate** (Privios style)
   - Zwarte fullscreen overlay
   - Centered logo + vraag
   - Twee duidelijke buttons
   - LocalStorage remember

2. **Product Card** (Hybrid)
   - Clean productfoto (Taylors)
   - Rating + reviews (Wijnvoordeel)
   - Member pricing (Taylors)
   - Hover effect subtle

3. **Product Detail** (Taylors base + Privios flair)
   - 2-kolom layout
   - Smaakprofiel indicators
   - Member vs regular pricing
   - Shopify checkout integratie

4. **Filter Sidebar** (Wijnvoordeel)
   - Land, Druivenras, Smaakprofiel
   - Checkbox + counts
   - Accordion style

5. **Hero Section**
   - Optie voor video (Privios) of carousel
   - Overlay tekst
   - Prominent CTA

### Unieke Vino per Lei Elementen
- "Per Lei" = "Voor Haar" - vrouwelijke, elegante touch
- Mogelijk roze/rosé accent kleur
- Persoonlijke cadeauservice highlighten
- Nederlandse taal, maar internationale uitstraling
